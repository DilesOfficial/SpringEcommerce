## 📖 What is this?
Diles E-Commerce is a state-of-the-art web application that bridges the gap between top-tier tech enthusiasts and premium products. Powered by a robust **Spring Boot (Java)** backend and a blazing-fast **React / Vite** frontend, this platform boasts a radically modern, light-themed immersive user experience beautifully styled with **Tailwind CSS**.

## ✨ Key Features
- **Live Seamless Search Engine**: Find your gear instantly with dynamic, real-time dropdown search suggestions showcasing product thumbnails.
- **Modern Minimalist UI**: High-end styling elements featuring floating background animations, glassmorphic navbars, and gradient highlight text.
- **Fully Responsive Layout**: Flawless transitions and card layouts adapting gracefully from mobile devices to desktop monitors.
- **RESTful Architecture**: Secure, fast, and optimized Spring Boot API endpoints for full-stack data fetching.
- **Complete Product Management**: Add, Update, Search, Purchase, and Delete gear seamlessly.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Java 21+ & Maven 
- MySQL Database

### Running the Frontend
1. Clone the repository and install dependencies
   ```bash
   npm install
   ```
2. Start the Vite development server
   ```bash
   npm run dev
   ```

### Running the Backend
1. Configure your MySQL credentials in `application.properties`.
2. Run the Spring Boot application using Maven or your IDE (IntelliJ/Eclipse).

## 🌍 Environment Setup
Ensure you configure the `.env` file in the frontend root to point to your Spring Boot API:
```env
VITE_BASE_URL=http://localhost:8080
```
